Audiobook Binder

An audiobook binder is a program that takes mp3 files and converts and combines them to m4b. I could not find any program that did all the things i needed so i wrote this simple but powerfull fully parallelized audiobook binder.

  • Parallelized MP3 to M4B conversion
  • Set chapters from either a CUE sheet or chapterized MP3 files
  • Sort input files by ID3v2 track number if available, alphabetically if not

The program is open source and distributed under the GPL-3.0 licence.
