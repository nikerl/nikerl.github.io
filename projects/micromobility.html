Västtrafik Micromobility

We have made a trip planner that combines existing public transport in Gothenburg with alternative solutions such as rental bicycles and electric scooters. The purpose is to make it easy for the user to search for a trip with the different options in the same view.

The goal is for public transport to attract more users if journeys are made easier, for example by cycling part of the route to avoid having to change trams, or cycling part of the route to get to their intended destination quicker.

This project was built with Python and Flask for the backend with a simple HTML/Javascript frontend. It was made in collaboration with Västtrafik and Chalmers University of Technology.
